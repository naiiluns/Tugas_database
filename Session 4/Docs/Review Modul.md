# Pertemuan 1

Modul ini membahas mengenai pengenalan dasar MariaDB sebagai sistem manajemen basis data, termasuk cara membuat, menampilkan, dan memilih database. Kita diperkenalkan pada perintah dasar seperti CREATE DATABASE, SHOW DATABASES, dan USE, sebagai langkah awal untuk memahami cara kerja lingkungan MariaDB.

---

# Pertemuan 2

Modul ini membahas mengenai pembuatan tabel dalam MariaDB, mulai dari menentukan nama tabel, kolom, serta tipe data yang sesuai. Selain itu, dijelaskan juga perintah CREATE TABLE, DESCRIBE, dan DROP TABLE untuk memahami struktur serta pengelolaan tabel di dalam database.

---

# Pertemuan 3

Modul ini membahas mengenai cara memasukkan data ke dalam tabel menggunakan perintah INSERT INTO. Kita  belajar menambahkan satu atau beberapa baris data sekaligus serta memeriksa hasil input menggunakan perintah SELECT * FROM nama_tabel;.

---

# Pertemuan 4

Modul ini membahas mengenai manipulasi data yang sudah ada di dalam tabel dengan menggunakan perintah UPDATE, DELETE, dan SELECT yang dikombinasikan dengan klausa WHERE. Selain itu, kita juga diperkenalkan dengan operator logika seperti AND, OR, dan LIKE untuk memfilter data sesuai kebutuhan.

---

# Pertemuan 5

Modul ini membahas mengenai penyaringan dan pengurutan data menggunakan klausa WHERE serta ORDER BY. Kita mempelajari operator perbandingan (=, >, <, LIKE) untuk mencari data tertentu, dan mempraktikkan pencarian buku berdasarkan penulis agar hasil tampil lebih relevan dan terurut rapi.
